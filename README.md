# 🚕 WOO 클론코딩

**제작기간** : 2024.08 (2일) <br/>
**기여도** : 퍼블리싱 <br/>
**사용한 언어**: HTML, CSS <br/>
**URL** : https://woo-beta.vercel.app/

WOO는 정확히 말하자면 클론코딩이라기보다는 학원 수업 진도 중에 선생님과 함께 작업한 프로젝트이다. 이 프로젝트를 통해서 **vw**와 **vh**를 통한 **반응형 디자인**, 다양한 **호버효과**와  **무한 애니메이션**에 대한 개념을 더 정확히 할 수 있었고 **파비콘**을 생성하는 법을 익혔다.
